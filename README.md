# Simulation of Robots course project

## Installation guide

1) create or use environment with MuJoCo
2) clone [loco-mujoco](https://github.com/robfiras/loco-mujoco)
3) `cd loco-mujoco && pip install -e .`
4) `pip install jax["cuda12"]` - for GPU
5) `pip install stable-baselines3`
